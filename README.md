🍔 Mini Site Web – Burger King
=================================

Un mini‑projet front‑end reproduisant une page d’inspiration Burger King, avec une interface moderne, responsive et fidèle à l’univers visuel de la marque.

📌 Présentation
=================

Ce projet est un mini site web statique réalisé dans le cadre de mon apprentissge personnelle en développement web.
L’objectif était de recréer une page publicitaire Burger King en travaillant :

====================================================================================================
la structure HTML

la mise en page et le design en CSS

l’intégration d’images et d’éléments graphiques

la cohérence visuelle d’une marque existante
====================================================================================================
Ce projet me permet d’améliorer mes compétences en intégration web et en conception d’interfaces.

🎨 Fonctionnalités
====================

Page d’accueil inspirée d’une campagne Burger King

Mise en avant d’une varieté de produits (Menu au choix pour ce régaler)

Boutons interactifs (ex : Click & Collect, Découvrez-le !)

Design responsive adapté aux écrans modernes
Respect de la charte graphique (couleurs, typographies, ambiance)

🛠️ Technologies utilisées
==========================
HTML5
CSS3

📁 Structure du projet
=======================
Code
/site_web_burger_king
   /images

   README.md

🚀 Objectif du projet
======================
Ce mini‑site a été créé pour pratiquer l’intégration web, améliorer ma maîtrise du front‑end et apprendre à reproduire une interface professionnelle à partir d’une image ou d’un concept visuel.
